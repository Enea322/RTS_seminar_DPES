# Topic - Dynamic Priority Exchange Server

## Overview
The Dynamic Priority Exchange (DPE) server is an aperiodic service technique proposed by Spuri and Buttazzo. It is a fixed-priority service algorithm, whose objective is to reduce the average response time of aperiodic requests without compromising the schedulability of hard periodic tasks.
